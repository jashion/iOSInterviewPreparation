# iOSInterviewPreparation

> 个人应聘iOS开发工程师可能会面对的一些问题，我主要把它们分成了四部分，技术，项目，管理和开放性问题。其中技术是基础，占比最高，其次是项目经验，管理是加分项，分项目管理和人的管理，最后开放性问题就比较杂，大致分成关于个人的问题，比如个人职业规划等等，关于公司的问题，比如为什么选择我们公司等等。
